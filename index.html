Rubin 세대 직접 경쟁자. AMD가 “engineering projection”이라고 밝힌 수치라 확정 스펙은 아님.

세대가 바뀌면서 비교 단위가 FP16 → FP8 → FP4로 한 칸씩 내려갔다. 칩의 절대 성능보다 “어떤 정밀도에서, 어떤 메모리 대역폭과 통신으로 묶이는가”가 진짜 비교축이 됐다.

※ 수치 출처 — NVIDIA / Google Cloud / AWS Neuron / AMD 공식 자료. Rubin · MI400 등 일부는 preliminary projection.
※ FP4 / FP8 PFLOPS는 dense 기준. 단위·범위는 각 벤더 발표 기준을 따름.
※ 비교표는 per-chip(or per-GPU)으로 정규화. Scatter는 GB300 NVL72(랙 스케일) 제외.
